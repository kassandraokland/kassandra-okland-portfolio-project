Healthy Inspiration

Healthy Inspiration is the food blog I made for the Project Exam.

List of changes

  • I added some div-tags to the blog posts so that I could display the content that I added from the WP REST API as flexbox.
  • I added the modal function to the images, because didn't get the time to do that while I worked on the Project Exam.
  • The font sizes needed some more testing and debugging. The main text was a little too small certain places, and the logo was a little too big on the wider screens.

Community Science Museum

Community Science Museum is the website I made for Semester Project 1. This is the project that needed the most updates and tweaks.

List of changes

  • I made the h1 smaller, because it was too big on screens over 650px and that made it look a little desperate.
  • I added the modal function to the images, because I ran out of time
  • I added call to action buttons to all landing pages that didn't already have one. The pages got one read more button and one buy tickets. The read more button brings you down the same page to the main content, while the buy ticket button brings you to a buy ticket page. The buy ticket page is also new.
  • I added more hover and action styles, for instance to the cards of the menu page.
  • I finished the content of the specific pages. Earlier I had just made the content for the first specific page.
  • I added some missing alt tags on the explore page (for researchers picture) and on the index page (visiting prof. image).
  • I transformed some h1 into other heading types, because I had multiple h1 tags on one page, and I have learnt now that you are not supposed to do that.
